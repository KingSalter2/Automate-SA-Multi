# EB Motors

Premium pre-owned and new vehicles dealership management system.

## Project Info

**Website**: https://ebmotors.co.za
**Built by**: [Rabbit 365](https://www.rabbit365.co.za)

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
